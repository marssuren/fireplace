#### 主要翻译修正：

1. **TID_711**: Ozumat → **厄祖玛特**

2. **TID_712**: Neptulon the Tidehunter → **猎潮者耐普图隆**

    - "奈普图隆" → "耐普图隆"
    - "手部" → "耐普图隆之手"
    - "它们攻击" → "手攻击"

3. **TSC_032**: Blademaster Okani → **剑圣奥卡尼**

    - 补充了"当本随从存活时"的条件

4. **TSC_067**: Ambassador Faelin → **费林大使**

    - "置入...底部" → "置于...底"
    - "巨型随从" → "巨型随从牌"

5. **TSC_641**: Queen Azshara → **艾萨拉女王**

    - "持有该牌时施放过 3 个法术" → "本牌在你手中时施放过三个法术"
    - "远古遗物" → "远古圣物"
    - "选择一个" → "选择一项"

6. **TSC_649**: Ini Stormcoil → **伊妮·积雷**

    - "该随从的复制" → "它的...复制"
    - 调整标点符号

7. **TSC_908**: Sir Finley, Sea Guide → **海中向导芬利爵士**
    - "与...交换" → "和...交换"
    - "底部" → "底"
